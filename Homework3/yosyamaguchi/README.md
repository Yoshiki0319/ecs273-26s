# Stock Data Visualization

This project is a stock data visualization application built with React and Vite (using D3.js).

## Prerequisites

Make sure you have the following environments installed:
- Node.js (v18.x or higher recommended)
- npm (v9.x or higher recommended)

## Getting Started

Follow the steps below to start the application on your local environment and view it in your browser.

### 1. Install Dependencies

Navigate to the `yosyamaguchi` directory in your terminal and run the following command:

```bash
npm install
```

### 2. Start the Development Server

Once the installation is complete, start the development server with the following command:

```bash
npm run dev
```

### 3. View in Browser

When the server starts, a local URL (e.g., http://localhost:5173 or http://localhost:3000) will be displayed in the terminal. Open that URL in your browser to view the application.
